# github-page
criando site
nessa area esta desenvolvendo o porifolio pessoal e profissional do bootcamp na estapa intermediaria

O repositório no GitHub serve para armazenar, organizar e versionar projetos acadêmicos e pessoais de forma profissional. Conforme o PDF da atividade, o objetivo é criar o “PortfolioHUB”, utilizando Git e GitHub para aplicar boas práticas de versionamento, organização e documentação dos projetos. Além disso, o repositório funciona como um portfólio online, permitindo compartilhar os trabalhos, publicar uma página no GitHub Pages e integrar os projetos ao LinkedIn, desenvolvendo habilidades importantes utilizadas no mercado de tecnologia.
